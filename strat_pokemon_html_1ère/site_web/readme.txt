Je suis Mael, élève de 1ère générale au lycée Lucie Aubrac. J'ai développé l'entièreté de ce projet. J'ai testé les 5 pages du site avec 3 navigateurs différents (brave, chrome et edge) et tous fonctionnent.
Ce projet est un site web sur le thème de la stratégie en combats pokémon. Il traite des bases à connaitre pour débuter dans la stratégie pokémon.
Ce site est divisé en 5 pages + une feuille de style CSS :
- page d'accueil : menu fait avec une liste et des liens internes, 1 vidéo youtube intégrée avec balise iframe, 2 vidéos mp4 intégrées avec balise video
- page 1 : menu idem page d'accueil, 1er script intégré au code html*, images, 2ème script intégré au code html**
- page 2 : menu idem page d'accueil, images
- page 3 : menu idem page d'accueil, lien externe, images
- page 4 : menu idem page d'accueil, tableaux, image

*le script de la page 1 affiche une alerte lorsque l'image ou le gif d'un des 3 pokémon est cliquée. Cette alerte affiche des informations complémentaires sur le pokémon cliqué

**le 2ème script (en haut de page) affiche un diaporama affichant les symboles des 18 types, chacun pendant 1 seconde.
