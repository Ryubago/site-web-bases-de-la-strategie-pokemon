Connaitre les bases de la stratégie pokémon

Vous pouvez lire les étapes dans l'ordre que vous voulez

Mon but ici n'est pas de vous former parfaitement pour briller en compétition, mais de vous aider à mieux comprendre différentes choses complexes ou contrintuitives non citées dans les jeux.

Ici, vous apprendrez étape par étape les bases de la stratégie pokémon pour les jeux allant de la 1ère à la 9ème génération

Pour mieux comprendre quels jeux appartiennent à quelle génération, voici une liste de chaque jeu par génération :

4ème génération : Perle/Diamant/Platine, HeartGold/SoulSilver

5ème génération : Noir/Blanc/Blanc2/Noir2

6ème génération : X/Y, Rubis Oméga/Saphir Alpha

7ème génération : Soleil/Lune/Ultra-Soleil/Ultra-Lune, Let's go Pikachu/let's go Evoli

8ème génération : Epée/Bouclier/Couronne/Armure, Légendes Arceus, Diamant étincelant/Perle scintillante

9ème génération : Ecarlate/Violet/Turquoise/Indigo, Légendes Z-A
